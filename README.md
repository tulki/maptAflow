# maptAflow

**maptAflow** — это пространственная система управления задачами, предназначенная для плавного перехода от стратегического мышления к выполнению задач.

Система решает разрыв между:

Planning → Structuring → Execution

Большинство инструментов хорошо подходят для планирования, но создают трение на этапе выполнения задач.  
maptAflow фокусируется на том, чтобы сделать **исполнение задач максимально естественным и непрерывным**.

---

# Core Idea

В основе maptAflow лежит **пространственная карта задач**.

Каждая задача представлена узлом на бесконечном холсте. Узлы могут связываться между собой, формируя структуру задач и подзадач.

Навигация строится вокруг **пространственного перемещения**, а не вокруг списков или страниц.

---

# Three-Level Model

Система строится вокруг трёх уровней мышления.

## Global Level — Strategy

Глобальная карта проектов и направлений.

Пользователь видит общую структуру своих задач.

---

## Node Level — Tactics

Декомпозиция конкретной задачи.

Каждый узел может раскрывать отдельное дерево подзадач.

---

## Local Level — Execution

Минимальный интерфейс для выполнения задач.

Фокус на текущей работе без лишнего интерфейса.

---

# Current MVP

Сейчас реализуется базовый Graph Engine.

Реализовано:

- infinite canvas
- pan холста
- создание нод
- drag нод
- связывание нод
- направленные связи со стрелками
- правило одного родителя для ноды
